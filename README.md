# Yopass Tools

Набор утилит для разбиения большого текстового файла на части, загрузки частей в Yopass и последующего восстановления исходного файла.

## Состав

- `yopass_code.py` — разбивает файл на части, загружает каждую часть в Yopass и сохраняет ссылки.
- `yopass_decod.py` — читает ссылки из файла, скачивает и расшифровывает секреты, восстанавливая исходный файл.

---

## Требования

- Python 3.10+
- Доступ к `https://pass.rtlabs.ru`
- При необходимости — подключение к корпоративному VPN.

Скрипты самостоятельно устанавливают библиотеку **PGPy** во временный каталог, если она отсутствует.

---

# Создание ссылок

## Подготовка

По умолчанию исходный файл должен называться:

```text
text.txt
````

или можно указать любой другой файл при запуске.

## Запуск

Использовать файл `text.txt`:

```bash
python yopass_code.py
```

Использовать другой файл:

```bash
python yopass_code.py my_file.txt
```

---

## Что делает скрипт

1. Проверяет доступность `pass.rtlabs.ru`.
2. Разбивает файл на части по 50 строк.
3. Каждую часть локально шифрует.
4. Загружает её в Yopass.
5. Получает ссылку.
6. Записывает ссылку в файл `total`.
7. После успешного завершения удаляет временный каталог с частями.

---

## Результат

Будет создан файл:

```text
total
```

Пример содержимого:

```text
https://pass.rtlabs.ru/#/s/xxxxxxxxxxxxxxxx/aaaaaaaaaaaaaaaa
https://pass.rtlabs.ru/#/s/yyyyyyyyyyyyyyyy/bbbbbbbbbbbbbbbb
https://pass.rtlabs.ru/#/s/zzzzzzzzzzzzzzzz/cccccccccccccccc
```

Каждая строка содержит одну ссылку.

---

# Восстановление файла

По умолчанию скрипт читает ссылки из файла:

```text
total
```

и создаёт файл:

```text
restored.txt
```

## Запуск

```bash
python yopass_decod.py
```

или

```bash
python yopass_decod.py my_total --output result.txt
```

---

## Что делает скрипт

1. Проверяет доступность `pass.rtlabs.ru`.
2. Читает ссылки по одной.
3. Скачивает зашифрованное содержимое.
4. Расшифровывает его локально.
5. Добавляет данные в восстановленный файл.

При возникновении ошибки уже восстановленная часть файла сохраняется.

---

# Параметры

## yopass_code.py

| Параметр       | Описание                                           |
| -------------- | -------------------------------------------------- |
| `input_file`   | Исходный файл (по умолчанию `text.txt`)            |
| `--lines`      | Количество строк в одной части (по умолчанию `50`) |
| `--delay`      | Задержка между загрузками (по умолчанию `1`)       |
| `--output`     | Файл со ссылками (по умолчанию `total`)            |
| `--expiration` | Время жизни ссылки (`1h`, `1d`, `1w`)              |
| `--multiple`   | Создать многоразовые ссылки                        |

---

## yopass_decod.py

| Параметр     | Описание                                |
| ------------ | --------------------------------------- |
| `links_file` | Файл со ссылками (по умолчанию `total`) |
| `--output`   | Имя восстановленного файла              |
| `--delay`    | Задержка между скачиваниями             |

---

# Особенности

* Шифрование выполняется локально.
* В Yopass отправляется только зашифрованный текст.
* Ключ расшифровки содержится во фрагменте ссылки (`#/...`) и не передаётся серверу.
* По умолчанию создаются одноразовые ссылки сроком действия **1 час**.
* Временные файлы автоматически удаляются после успешной загрузки.
